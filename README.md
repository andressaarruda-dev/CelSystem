# 📱CelSystem 
> Gerencie sua assistência técnica de forma simples e orgânizada.
## Sobre o projeto
O CelSystem é um sistema de gestão para assistências técnicas de celulares, desenvolvido como projeto de longo prazo durante toda a minha graduação em Análise e Desenvolvimento de Sistemas. 
O objetivo é aplicar, na prática, os conhecimentos adquiridos em cada disciplina da faculdade, evoluindo continuamente o sistema até uma versão completa. 
------------------------------------------------------------------------------------------------------------------------------------------
## ✔ Objetivos
- Gerenciar clientes
- Gerenciar aparelhos
- Controlar ordens de serviços
- Acompanhar status de reparos
- Gerenciar encomendas
- Controlar estoques
- Gerenciar usuários
-  Disponibilizar acompanhamento para cliente
